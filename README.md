# 🚍 E-Transport System (Web & Mobile)

A full-stack transport management system designed for office and school transportation with real-time vehicle tracking.

## ✨ Features
- Driver & Passenger registration
- Real-time GPS tracking of vehicles
- Live location updates using Socket.io
- Attendance management (mark absence)
- Seat-sharing system for unused seats
- Payment handling for passengers
- Google AdSense integration

## 🛠 Tech Stack
- Frontend: React, React Native
- Backend: Node.js, Express
- Database: MongoDB
- Auth: Firebase Authentication
- Real-time: Socket.io
- Maps: OpenStreetMap

## 🚀 How It Works
- Drivers start a trip → passengers can track live location
- Passengers can mark absence → others can request the seat
- System updates in real-time using WebSockets
